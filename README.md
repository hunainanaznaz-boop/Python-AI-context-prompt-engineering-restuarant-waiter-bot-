# Python-AI-context-prompt-engineering-restuarant-waiter-bot-
Python-based AI Restaurant Waiter Bot with Gradio interface, menu validation, order management, billing, and receipt generation.

## Summary

AI Restaurant Waiter Bot is a Python-based chatbot with a Gradio interface that simulates a restaurant waiter. It greets customers, displays the menu, recommends food items, takes orders, validates menu items, calculates bills, and generates receipts.

## Features

- Customer Greeting
- Menu Display
- Food Recommendation
- Order Management
- Receipt Generation
- Gradio Interface
- Menu Validation

## Technologies Used

- Python
- Gradio
- Regex
- Prompt Engineering

## Run Project

```bash
pip install gradio
python app.py
```
